Este programa procesa una lista de imágenes, realiza la detección de bordes en cada una y muestra tanto la imagen original como la imagen procesada con bordes. Además, acumula y muestra el histograma de todas las imágenes procesadas.

Requisitos
Antes de ejecutar el programa, asegúrate de tener instaladas las siguientes bibliotecas:

imageio
matplotlib
numpy
scipy
os (biblioteca estándar de Python)
Puedes instalarlas utilizando pip:

bash
Copiar código
pip install imageio matplotlib numpy scipy
Uso
Coloca las imágenes que deseas procesar en el mismo directorio que el script, o especifica la ruta correcta de las imágenes en la lista image_files.
Ejecuta el script.
